## 服务端Node + 前端 js

* 安装express（初级使用） ✔
* 安装express-handlerbars  ✔
* 安装sass  ✔
* 安装gulp（最开始配合compass使用） ✔
* 开始使用gulp ✔
* express高级使用，前端架构逐步生成
* jquery/angular选型，使用
* mongoose


tips:
* 个人项目中将会使用gulp，而团队项目中将会使用grunt
* 个人项目将会使用jquery+require或者angular

blog规划：
1、结构规划：
左侧导航栏,需要响应式，768px以上显示，768px以下以按钮形式展示

2、中间图片，响应式

3、列表，每一项需要能显示简洁或者不显示

a、首页之类的列表，需要展示的信息比较多

b、普通的列表，不需要展示简介

c、归档之类的列表，不需要展示太多信息
